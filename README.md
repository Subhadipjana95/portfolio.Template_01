# Minimal Portfolio

A tastefully minimal portfolio template built for developers. Clean, fast and ready to make your own.

## Stack

- **Next.js 16** — App Router
- **React 19** — with full concurrent features
- **Tailwind CSS v4** — utility-first styling
- **Motion** — smooth, physics-based animations
- **shadcn/ui** — accessible, composable UI primitives
- **Remix Icons** — crisp developer-friendly icon set
- **TypeScript** — end-to-end type safety

## Features

- Dark/light mode toggle
- Minimal, distraction-free layout
- Mobile responsive out of the box
- Easy to extend with your own sections

## Getting Started

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000) and start editing the files of `Data/[section].data.tsx`.

## Customization

The template is intentionally minimal — drop in your own sections, swap out colors in `globals.css`, and replace the placeholder content with your work. The `components/` directory has reusable pieces you can pull from directly.
